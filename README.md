## ✨ Features

- 🎨 **Material 3 Design** with custom dark/light theme
- 🌐 **Responsive Layout** — mobile, tablet, desktop
- 🌍 **Bilingual** — Turkish / English support
- ⚡ **Fast Performance** — optimized for Web
- ♿ **Accessible** — WCAG 2.1 AA compliant
- 🔧 **Maintainable** — clean architecture, production-ready code

## 🛠 Tech Stack

| Category | Technology |
|----------|-----------|
| Framework | Flutter Web |
| State Management | Riverpod |
| Routing | GoRouter |
| Theming | Material 3 + Google Fonts |
| Icons | Material Icons |
| Storage | Shared Preferences |
